# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project builds a Machine Learning model to classify Iris flowers into three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

The model learns from flower measurements such as sepal length, sepal width, petal length, and petal width and predicts the correct flower species.

This is one of the most widely used beginner-to-intermediate Machine Learning classification projects and demonstrates the complete ML workflow from data exploration to model evaluation.

The project uses the Iris Dataset available in Scikit-Learn.

Dataset Information:

- 150 Samples
- 4 Features
- 3 Classes

Features:

1. Sepal Length
2. Sepal Width
3. Petal Length
4. Petal Width

Target Classes:

- Setosa
